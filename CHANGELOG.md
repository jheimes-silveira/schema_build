## 0.1.0

* Initial release of `schema_build`.
* **Server-Driven UI (SDUI)** engine with JSON schema support.
* **Visual Builder**: Drag-and-drop editor for layout creation.
* **Reactive Architecture**: Push-based state management via `SchemaComponentController`.
* **Extensible System**: Easy registration of custom components.
* **Zero-Rebuild Reactivity**: Logic observation via `updateExternalState`.
* **Async Lifecycle**: `onInit` hook for component setup.
